üé® Design Philosophy

Consistent Visual Language: Instead of using different colors for each game mode, 1280 Trivia uses a unified Netflix-inspired dark interface that provides a cohesive, professional experience across all question types.

What Makes Each Mode Unique: Game modes are differentiated by their content (chat receipts vs trivia vs polls), mechanics (voting vs correct answers vs minigames), and scoring systems (points for accuracy vs points for consensus).

Inspired By: Jackbox Games, Netflix UI, modern streaming interfaces. The dark theme reduces eye strain during gameplay and creates an immersive party game atmosphere.
